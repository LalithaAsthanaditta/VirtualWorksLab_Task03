# ☕ Coffee Rating Application

A simple web application built using **Python Flask, HTML, CSS, and SQLite** that allows users to vote for their favorite coffee. The application stores vote counts in a database and updates them dynamically.

## Features

- Display a list of coffee varieties
- Vote for your favorite coffee
- Real-time vote count updates
- Database integration using SQLite
- Simple and user-friendly interface

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- SQLite

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd CoffeeRatingApp
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
